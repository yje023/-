# 全区单位考核系统 — 需求文档 v1.1

## 项目背景
全区单位考核系统，用于政府对各单位的年度考核管理。支持主考单位发布考核任务、被考核单位在线填报、主考单位打分评估。

## 系统角色
- **发布单位**：拥有所有权限，可进行系统管理和考核方案制定
- **主考单位**：参与考核评分，对分配给自己的被考核单位进行打分
- **被考核单位**：接收考核任务，在线填报完成情况
- **自定义角色**：管理员可创建角色并分配菜单权限

## 功能需求

### 一、系统后台管理

#### 1. 机构管理
- 创建、编辑、删除组织机构（树形结构，只需名称）
- xlsx 批量导入/导出组织机构
- 批量删除组织机构
- 搜索组织机构

#### 2. 单位管理
- 在组织机构下创建单位（只需名称）
- 编辑、删除单位
- 将单位移动到其他组织机构
- xlsx 批量导入/导出单位
- 批量删除单位
- 单位详情显示：单位名称 + 所属组织机构
- 搜索单位

#### 3. 用户管理
- 创建单位时自动生成初始账号：用户名=单位名称+账号，密码=csma123
- 在单位下手动创建用户（用户名+密码）
- 编辑、删除用户
- 用户详情显示：用户名、密码、角色、所属单位
- 登录后强制修改初始密码，所有用户均可修改密码
- 用户在"主考单位"和"被考核单位"身份间切换
- 搜索用户

#### 4. 角色管理
- 创建、编辑、删除角色
- 角色配置：名称 + 后台管理菜单权限 + 生产端菜单权限
- 系统预置"发布单位"角色（全权限）
- 搜索角色

### 二、生产端数据管理

#### 5. 考核方案管理
- 创建、编辑、删除考核方案
- 方案信息：名称、考核年度、起止时间、创建日期
- 评价维度管理（新增/编辑/删除），所有维度分值合计=100
- 系统预置"单位实际考核"评价维度
- 在"单位实际考核"下创建考核维度（只需名称）
- 从组织机构树多选添加主考单位，支持搜索勾选
- 创建被考核单位分组，为分组匹配考核维度并赋权重（权重合计=100%）
- 考核方案下发

#### 6. 考核任务管理
- 考核任务批量 xlsx 导入
- 编辑、删除、审核考核任务
- 任务信息：考核维度、重点工作名称、主要任务名称、评分说明、晾晒周期（月度/季度/半年度/年度）、评价部门（来自主考单位）
- 导入任务自动分拨到各单位账号
- 多维度导出考核任务书

#### 7. 填报打分（1.0 包含）
- 被考核单位：查看任务 + 在线填报完成情况 + 提交
- 主考单位：查看待打分任务 + 在线打分 + 提交
- 得分计算：考核维度权重折算 × 单位实际考核分值占比

### 三、后续版本规划
- 其他评价维度（民主测评、领导评价等）独立菜单
- 特定单位直接打分
- 角色权限细化到其他评价维度
- 数据统计与报表

## UI设计原则
- 简洁直观，淡蓝色主色调
- 交互简单，易操作
- 响应式布局，适配不同屏幕
