# 全区单位考核系统 — 设计规范 v1.1

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色调 | #409EFF | Element Plus 默认蓝 |
| 主色浅 | #ECF5FF | 选中背景、标签 |
| 侧边栏背景 | #304156 | 深色侧边栏 |
| 侧边栏文字 | #BFCBD9 | 侧边栏菜单文字 |
| 页面背景 | #F0F2F5 | 主内容区背景 |
| 成功 | #67C23A | 操作成功提示 |
| 警告 | #E6A23C | 警告提示 |
| 危险 | #F56C6C | 删除、错误提示 |

## 布局规范
- 侧边栏宽度：220px
- 顶部栏高度：60px
- 内容区 padding：20px
- 卡片背景色：#FFFFFF，圆角 4px

## 字体
- 系统默认使用"Microsoft YaHei"（Windows）
- 代码/数字使用等宽字体

## 组件规范
- 表格：带边框、斑马纹、分页
- 表单：标签右对齐，输入框宽度 200-400px
- 弹窗：宽度根据内容 400-800px
- 按钮：主操作蓝色、危险操作红色
- 树形控件：用于组织机构，支持展开/折叠/高亮

## 交互规范
- 删除操作需二次确认
- 批量操作前需勾选数据
- 表单提交后关闭弹窗并刷新列表
- 操作成功/失败有提示消息
- 加载中状态显示 loading
